== Description ==

Display a simple Google Maps via a WordPress shortcode for posts, pages or widgets.

Below are some of its features:

 * Simple shortcode control
 * API key compatible
 * Marker Pin
 * Zoom level
 * Info Window

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/yumjam-simple-map-shortcode` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->YumJam Map screen to configure the plugin

== Frequently Asked Questions ==

= What does YumJam Simple Map Shortcode Do? =

Allows you to easily display a simple Google Maps via a WordPress shortcode on posts, pages and widgets, great for contact pages.

= Can I customise the map? =

Yes, some simple customisation is available
